Cómo usar la aplicación

Modo distancia

  1. Haz clic en el mapa para fijar el punto inicial.
  2. Haz clic de nuevo para fijar el objetivo.
  3. La distancia se calcula automáticamente.

Modo mortero

  1. Activa el interruptor de modo mortero.
  2. Coloca primero la posición del mortero.
  3. Después coloca el objetivo.
  4. El HUD mostrará distancia, ángulo, tiempo y estado del disparo.

Compartir e instalar

  1. Usa Compartir para enviar un enlace con mapa, tema y modo activos.
  2. Usa Instalar cuando el navegador permita añadir la app al dispositivo.
  3. En iPhone o iPad, abre Safari y usa Compartir → Añadir a pantalla de inicio.

Accesos móviles

  1. Usa la barra flotante inferior para abrir controles, dibujo, HUD, compartir o ayuda.
  2. Contrae el HUD cuando necesites más espacio para el mapa.
  3. El encabezado permanece fijo para cambiar mapa, modo y tema sin perder contexto.

Herramientas de dibujo

  1. Abre el panel Dibujar.
  2. Selecciona una herramienta y un color.
  3. Para texto, escribe el contenido y haz clic sobre el mapa.
  4. Para borrar, elige Borrador y haz clic sobre un dibujo.

Atajos

  • R reinicia medición y encuadre.
  • C limpia la medición actual.
  • D abre o cierra el panel de dibujo.
  • H abre la ayuda.
  • M alterna el modo mortero.
  • S comparte la app.
  • E exporta la vista actual.
  • Esc cierra paneles.
