Guía rápida

Cómo usar la aplicación

Nueva navegación

  1. Usa Mapa para cambiar escenario y confirmar el contexto activo.
  2. Usa Tiro para alternar entre medición y mortero, además de reiniciar o limpiar.
  3. Usa Dibujo para abrir la capa táctica y acceder a herramientas visuales.
  4. Usa Acciones para compartir, exportar o instalar.
  5. Usa Más para ayuda rápida, tema y atajos.

Modo distancia

  1. Haz clic en el mapa para fijar el punto inicial.
  2. Haz clic de nuevo para fijar el objetivo.
  3. La distancia se calcula automáticamente y se resume en el HUD inferior.

Modo mortero

  1. Activa el interruptor de modo mortero en la sección Tiro.
  2. Coloca primero la posición del mortero.
  3. Después coloca el objetivo.
  4. El HUD mostrará distancia, ángulo, tiempo y estado del disparo.

Dibujo táctico

  1. Abre la sección Dibujo y entra a las herramientas.
  2. Selecciona herramienta y color.
  3. Para texto, escribe el contenido y haz clic sobre el mapa.
  4. Para borrar, elige Borrador y haz clic sobre un dibujo.

Compartir e instalar

  1. Usa Compartir para enviar un enlace con mapa, modo y tema activos.
  2. Usa Exportar vista para generar una imagen del mapa actual.
  3. Usa Instalar cuando el navegador permita añadir la app al dispositivo.

Atajos

  • R reinicia medición y encuadre.
  • C limpia la medición actual.
  • D abre o cierra el panel de dibujo.
  • H abre la ayuda.
  • M alterna el modo mortero.
  • S comparte la app.
  • E exporta la vista actual.
  • I lanza la instalación cuando está disponible.
  • Esc cierra paneles.
